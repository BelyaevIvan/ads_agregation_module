.Заголовок `Authorization` не передан:
[source, JSON]
....
HTTP/1.1 401 Unauthorized

{
  "statusCode": 401,
  "url": "/api/v1/users/me",
  "message": "отсутствует токен авторизации",
  "date": "2026-04-17T14:32:00Z"
}
....

.Заголовок `Authorization` имеет некорректный формат:
[source, JSON]
....
HTTP/1.1 401 Unauthorized

{
  "statusCode": 401,
  "url": "/api/v1/users/me",
  "message": "неверный формат заголовка Authorization",
  "date": "2026-04-17T14:32:00Z"
}
....

.JWT невалидный или просроченный:
[source, JSON]
....
HTTP/1.1 401 Unauthorized

{
  "statusCode": 401,
  "url": "/api/v1/users/me",
  "message": "невалидный или просроченный токен",
  "date": "2026-04-17T14:32:00Z"
}
....

.Пользователь, связанный с токеном, удалён из БД:
[source, JSON]
....
HTTP/1.1 404 Not Found

{
  "statusCode": 404,
  "url": "/api/v1/users/me",
  "message": "пользователь не найден",
  "date": "2026-04-17T14:32:00Z"
}
....

.Непредвиденная ошибка на стороне сервера:

....

{
  "statusCode": 500,
  "url": "/api/v1/users/me",
  "message": "внутренняя ошибка сервера, попробуйте позже",
  "date": "2026-04-17T14:32:00Z"
}
....
