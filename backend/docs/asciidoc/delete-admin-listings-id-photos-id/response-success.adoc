.Пример ответа при удалении обложки (следующее фото автоматически становится обложкой):
[source, JSON]
....
HTTP/1.1 200 OK

{
  "deleted_photo_id": "c3d4e5f6-a7b8-4c9d-8e0f-1a2b3c4d5e6f",
  "new_cover_id": "d4e5f6a7-b8c9-4d0e-9f1a-2b3c4d5e6f7a"
}
....

.Пример ответа при удалении обычного (не обложечного) фото либо при удалении последнего фото в объявлении:
[source, JSON]
....
HTTP/1.1 200 OK

{
  "deleted_photo_id": "c3d4e5f6-a7b8-4c9d-8e0f-1a2b3c4d5e6f",
  "new_cover_id": null
}
