= API BrandHunt — список методов
:toc-title: Содержание
:table-stripes: even
:table-caption!:

Документация к REST API сервиса **api-service** (Go, стандартная библиотека `net/http`).

Базовый URL: `/api/v1`.

Защищённые методы требуют заголовок `Authorization: Bearer <jwt>`.
